# AI Agent

## CLI LLM agent using Google Gemini API

### Capabilities

#### Chatbot using latest Google Gemini models

#### Read/write to files (software, csv, MD, etc.)

#### Author test cases

#### Execute software and tests

### > Warning

> This software is designed to only operate in the directory specified by the user
> Prompt injection safety cannot be guaranteed, make sure you trust any skills or context files
> This software is not meant to replace Claude Code or OpenCode in a production environment
